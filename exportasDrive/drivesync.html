📖 How to set up DriveSync

DriveSync uses a Google Apps Script as your personal backend. Your data goes directly to your own Google Drive — no third-party servers, ever.

1

Create the Apps Script project

Set up the backend code in your Google account

①
Go to script.google.com and click New project
②
Delete all existing code in the editor and paste the entire contents of Code.gs (the file provided alongside this HTML)
③
At the very top of the file, find the CONFIG block and set your own username and password:
USERNAME : "admin",  ← change to anything you want
PASSWORD : "yourpassword123",  ← change to a strong password
④
Click Save (Ctrl+S / Cmd+S) and give your project a name like DriveSync
💡
These credentials are set by you — DriveSync has no signup system. Whatever you put here is what you type on the login screen.
2

Deploy as a Web App

This generates the Script URL you'll paste into the Setup screen

①
In the Apps Script editor, click Deploy (top right) → New deployment
②
Click the gear icon ⚙ next to "Select type" and choose Web app
③
Set these two settings exactly — both are required for DriveSync to work:
Execute asMe
Who has accessAnyone
④
Click Deploy → click Authorize access → sign in with Google → click Allow
⑤
Copy the Web App URL — it looks like:
https://script.google.com/macros/s/AKfycb.../exec
⑥
Paste this URL into the Setup screen of this app
⚠️
Every time you change Code.gs, you must redeploy or the old code keeps running:
Deploy → Manage deployments → click the pencil ✏️ → change version to New version → Deploy.
3

Connect & log in

Paste your URL and sign in

①
On the Setup screen, paste your Web App URL and click Continue
②
On the Login screen, enter the username and password you set in Step 1 above
③
After logging in, go to Settings → Backup URL to Drive — this saves your Script URL inside your own Drive so you never lose it
💡
Your Script URL is saved in your browser after first login. You only need to enter it once per device.
?

Forgot your URL or password?

How to recover everything

🔗 Forgot the Script URL
①
Open Google Drive and search for the folder LocalStorageBackups
②
Open the file _config.json — it contains your Script URL, username, and the save date
🔑 Forgot the password
①
Go to script.google.com and open your DriveSync project
②
Look at the top of Code.gs — your USERNAME and PASSWORD are written there in plain text
③
Change them if needed, then redeploy: Deploy → Manage Deployments → Edit → New version → Deploy
🔒 How your data stays private
✓
Your data flows directly: browser → your Apps Script → your Google Drive. No third-party servers.
✓
The Apps Script runs under your Google account — backups are in your personal Drive.
✓
Your password is never stored permanently — it's held in memory only during your active session.
✓
Backups are plain JSON files in LocalStorageBackups/{AppName}/ — readable and downloadable directly from Drive.
